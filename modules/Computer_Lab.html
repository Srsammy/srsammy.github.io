COMPUTER LAB

MODULE OVERVIEW

The computer lab module is a dedicated space with multiple workstations where people can use computers to complete tasks, build skills, and work on creative or technical projects. It provides access to important tools and software for activities like coding, 3D modeling, research, and digital design. This lab is especially helpful for people who may not have their own computer or internet access at home. Whether someone is learning something new, working on a project, or just needs a quiet place to use technology, the computer lab offers a supportive and accessible space for everyone.

HOW IT WORKS

When someone comes into the computer lab, they can choose an open workstation. Each station includes a computer, monitor, keyboard, and mouse. To get started, the user turns on the equipment and logs into a shared account that is managed by the makerspace team. This makes it easier to keep the systems working properly and safe for everyone. Once logged in, users can open the programs they need—such as a word processor, design software, or a web browser—and begin their work.

USES AND PROJECTS

A computer lab can be used for many different projects depending on the available software. One common use is 3D modeling, where users create digital designs for objects they can later print using a 3D printer. Coding is another useful skill that can be practiced in the computer lab using free online resources like Free Code Camp. Users can also research information, write reports, and edit digital artwork using design software.
