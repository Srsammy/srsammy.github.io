CODING CLASS

MODULE OVERVIEW

The coding class module gives users the chance to learn how to code using free online lessons. It's a great way to explore web development, build digital skills, and create real coding projects. This module is designed for beginners with no experience as well as for those looking to improve their skills. It's an easy and cost-free way to start learning a valuable, in-demand skill.

HOW IT WORKS

To use this module, users simply log into a computer in the computer lab and visit freecodecamp.org. From there, they can choose a lesson and follow step-by-step tutorials in coding topics like HTML, CSS, and JavaScript. Each lesson includes small, guided projects that teach users how to write and test code in a simple and clear format. No prior experience is needed, and users can move at their own pace.

TOOLS AND REQUIREMENTS

This module requires access to a computer lab with internet access. No additional equipment, software, or materials are needed. Free Code Camp runs entirely through a web browser and is free to use. This module is very compatible with the computer lab module that we also have recommendations for.

USES AND PROJECTS

This module helps users build real coding skills while working on fun and useful projects. For example, one of the beginner lessons teaches how to make a custom coffee shop menu using HTML. Users learn how to write and organize content, format text, and change colors. As they progress, they can work on more advanced projects like web pages, calculators, and personal portfolios.
