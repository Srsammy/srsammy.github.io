BASIC PLUMBING

MODULE OVERVIEW

Plumbing is an essential skill for maintaining and repairing water systems in homes and buildings. This module provides a hands-on learning experience, teaching users how to fix common plumbing issues like leaks, broken pipes, and toilet installations. By practicing with real tools and materials, users can develop the skills needed for basic plumbing repairs. Whether someone wants to fix a household issue or explore plumbing as a potential career, this module offers a practical starting point.

HOW IT WORKS

This module includes a practice setup that mimics real plumbing systems, including a training toilet and PVC pipes. Users will learn step-by-step how to connect pipes, seal leaks, and install plumbing fixtures. Guided instructions and video tutorials will help beginners understand the tools and techniques involved. Participants can start with simple tasks like applying plumber's putty and using PTFE tape, then move on to more advanced skills such as pipe cutting and thread sealing. Hands-on practice helps reinforce learning, making it easier to apply these skills in real situations.
