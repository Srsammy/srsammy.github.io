BASIC PLUMBING

MODULE OVERVIEW

Plumbing is the practice of fixing and maintaining systems that move fluid within a building. This module aims to help a person become familiar with basic plumbing work, such as fixing leaks, installing a toilet, and installing pipe sections, while introducing them to the various tools a plumber uses daily.

FUNCTION

This module will be similar to an apprentice program, where a person will be able to learn the basics of plumbing. They will learn useful skills like sealing leaks, cutting pipes, connecting plumbing fixtures, and using tools like wrenches and pipe cutters. They could also learn simple plumbing math, such as measuring pipe length and flow rate.

APPLICATION

This module is useful for training new plumbers, teaching DIY repairs, or providing hands-on experience in a learning space. It can also be used to help people understand how plumbing works, which can be helpful in emergencies or when transitioning into the workforce.
