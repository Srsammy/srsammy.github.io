NEEDLEWORK

MODULE OVERVIEW

The Needlework Module is designed to give people the chance to learn and practice needlework skills. Not only is this a fun activty, but also a useful one. Users can make their own clothes, blankets, and other items. For refugee camps in cold environments, this module can be used to make warm clothing and blankets.

TOOLS AND REQUIREMENTS

This module provides the tools and materials needed for needlework activities such as crocheting and knitting. Users can create handmade items like scarves, sweaters, hats, and more.
